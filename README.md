# Label Automate - File Comparison Tool

A web-based file comparison tool that highlights differences between documents.

## Features
- Compare PDF files side by side
- Compare text files with difference highlighting
- Visual diff highlighting with color coding
- Download comparison results
- Responsive design

## Technologies Used
- Python Flask
- HTML/CSS/JavaScript
- PDF.js for PDF rendering
- Bootstrap for styling

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

## Usage
1. Upload two files to compare
2. Click "Compare Files"
3. View differences with color highlighting
4. Download results if needed
